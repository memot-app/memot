# 大きな見出し（#）

> これは引用です。メモや補足、ひとことの強調に使えます。

## サブタイトル（##）

本文は通常の段落として表示されます。強調したいコードは `inline code` のようにバッククォートで囲みます。ここでは *これで太字風* と **こちらも太字** のどちらも太く見えるようにしています。

### 箇条書き

- シンプルなリスト
- 使い方を簡潔にまとめられます
- ネストは通常のMarkdownと同様に

1. 番号付きリスト
2. 手順の表記にも便利

### 画像の挿入

![サンプル画像](/og-image.png)

画像は角丸と軽い影を付けて読みやすく表示されます。大きすぎる場合は自動で調整されます。

### リンク

リンクはブランドカラーで表示され、下線が見やすい太さになっています。

[めもっと公式サイト](https://lp.memot.app)

### コードブロック

```ts
// TypeScriptの例
type User = {
  id: string;
  name: string;
}

export function greet(u: User) {
  return `Hello, ${u.name}!`;
}
```

`コピー` ボタンからコードをクリップボードに保存できます。
